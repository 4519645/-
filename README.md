## 1. 项目背景与目标
本项目使用经典的 `cars.json` 数据集，旨在通过交互式可视化手段，探索 1970 年至 1982 年间全球汽车工业的发展趋势。本系统不仅展示了车辆性能（马力与油耗）的静态分布，更通过多图联动技术，揭示了不同产地（Origin）在技术规格（气缸数）和时代演变（历年油耗）中的深层关联。

## 2. 视觉编码 (Visual Encoding) 设计
本系统由三个协同视图组成，采用了一致的色彩编码标准：**美国 (USA: 蓝色)、日本 (Japan: 橙色)、欧洲 (Europe: 红色)**。

*   **视图 A：性能分布图 (散点图)**
    *   **X 轴：** 马力 (Horsepower)，衡量动力性能。
    *   **Y 轴：** 每加仑行驶英里数 (MPG)，衡量燃油经济性。
    *   **视觉通道：** 圆点位置表示性能坐标，颜色区分产地。
*   **视图 B：硬件规格统计 (柱状图)**
    *   **X 轴：** 产地 (Origin)。
    *   **Y 轴：** 平均气缸数 (Mean of Cylinders)。
    *   **功能：** 作为主视图的明细补充，展示选中样本的硬件特征。
*   **视图 C：技术演进趋势 (折线图)**
    *   **X 轴：** 时间 (Year)。
    *   **Y 轴：** 平均燃油效率 (MPG)。
    *   **功能：** 展示随时间推移，各产地在节能技术上的进步曲线。

## 3. 交互技术实现 (Interaction Techniques)
本项目实现了 **"筛选 -> 联动 -> 发现"** 的完整交互链路：

1.  **跨维度刷选 (Brushing & Linking):**
    *   用户可以在散点图上通过鼠标拖拽进行“区间选择”。
    *   **联动效应：** 右侧的柱状图和下方的折线图会实时过滤数据，仅针对框选出的车辆重新计算平均气缸数和历史趋势。这允许用户分析特定性能范围（如“低马力高效率”）车辆的产地构成。
2.  **多选过滤 (Legend Selection):**
    *   点击图表上方的**Origin 图例**。
    *   **联动效应：** 散点图会高亮选中产地的点，同时将其他产地变淡（降噪），折线图也将仅保留选中产地的趋势线，便于进行产地间的横向对比。
3.  **动态提示 (Tooltip):**
    *   鼠标悬停在任何数据点上，均会弹出包含该车型详细参数（名称、马力、油耗、年份）的浮窗，辅助精确读取数值。

## 4. 数据发现与洞察
通过本系统的交互功能，可以观察到以下现象：
*   **产地差异：** 美国的车型普遍分布在散点图的右下方（高马力、低效率），而日本和欧洲车型集中在左上方。
*   **时代演变：** 通过折线图联动可以发现，1973 年和 1979 年两次石油危机后，所有产地的 MPG 均有显著提升，反映了汽车工业对外部环境变化的响应。
*   **技术路径：** 利用刷选功能锁定高 MPG 车辆，可以观察到这些车型绝大多数采用了 4 气缸引擎。
